## ⏱️ Web Stopwatch Challenge
Project ini adalah hasil dari coding challenge kecil-kecilan yang saya buat untuk melatih logika JavaScript. Intinya, ini adalah aplikasi pengukur waktu (stopwatch) berbasis web yang dibikin framework atau library pihak ketiga. Pure vanilla code!

## ✨ Apa Saja Fiturnya?

* Fungsi Utama Lancar: Tombol Start, Pause, dan Reset bekerja presisi tanpa kendala.
* Hitungan Akurat: Waktu tercatat detail sampai milidetik.
* Responsif: Tampilannya tetap rapi, mau dibuka lewat laptop ataupun dari HP.

## 🛠️ Dibangun Menggunakan

* HTML5 – Untuk bikin pondasi dan struktur halamannya.
* CSS3 – Biar tampilannya bersih, modern, dan enak dilihat.
* JavaScript (ES6+) – Otak utama di balik logika timer dan manipulasi DOM-nya.

## 🚀 Mau Coba Jalankan di Lokal?
Gampang banget, kamu enggak perlu install apa-apa:

   1. Clone dulu repository ini ke komputer kamu.
   2. Buka folder project-nya.
   3. Langsung klik dua kali (atau buka) file index.html di browser kesayangan kamu.
